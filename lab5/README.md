# Лабораторна робота 5 - Express.js сервер

## Мета роботи

- Ознайомитися з фреймворком Express.js для створення серверних веб-додатків на Node.js
- Навчитися створювати прості маршрути (routes)
- Повертати статичні сторінки та дані у форматі JSON
- Розуміти основи структури Express-додатка

## Опис програми

Сервер Express.js з наступними можливостями:

- **Маршрути:**
  - `/` - головна сторінка (HTML)
  - `/time` - показує поточну дату та час
  - `/time/json` - повертає дату та час у форматі JSON
  - `/settime` - зберігає введену користувачем дату та час у файл `custom.txt`

- **Логування:** Усі звернення до `/time` записуються у файл `log.txt` з відміткою часу

- **Статичні файли:** Розміщені у папці `/public`

## Структура проєкту

```
lab5/
├── app.js              # Основний файл сервера
├── package.json        # Залежності та конфігурація
├── log.txt             # Лог запитів до /time
├── custom.txt          # Збережені дати та час від користувачів
├── README.md           # Документація
└── public/             # Статичні файли
    ├── index.html      # Головна сторінка
    ├── style.css       # Стилі
    ├── script.js        # JavaScript для форми
    └── images/         # Зображення
```

## Встановлення та запуск

### Вимоги

- Node.js (версія 12 або вище)
- npm

### Кроки встановлення

```bash
# Перейти в папку lab5
cd lab5

# Встановити залежності
npm install

# Запустити сервер
npm start
```

Після запуску сервер буде доступний за адресою: `http://localhost:3000`

## Маршрути

### GET /

Головна сторінка з навігацією та формою для введення дати та часу.

### GET /time

Повертає HTML з поточною датою та часом у форматі:
```
Поточний час: 02.10.2025, 14:30:15
```

**Примітка:** Кожен запит логується у файл `log.txt`.

### GET /time/json

Повертає JSON з поточною датою та часом:
```json
{
  "date": "02.10.2025",
  "time": "14:30:15"
}
```

**Примітка:** Кожен запит логується у файл `log.txt`.

### POST /settime

Зберігає введену користувачем дату та час у файл `custom.txt`.

**Параметри запиту (JSON):**
```json
{
  "date": "2025-10-02",
  "time": "14:30:15"
}
```

**Відповідь:**
```json
{
  "success": true,
  "message": "Дата та час збережено успішно!"
}
```
