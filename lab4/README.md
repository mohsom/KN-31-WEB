# Лабораторна робота 4 - HTTP сервер на Node.js

## Мета роботи

- Ознайомитися з можливостями Node.js для створення веб-сервера
- Навчитися створювати простий HTTP-сервер без додаткових фреймворків
- Закріпити практичні навички роботи з модулями Node.js та маршрутизацією

## Опис програми

Сервер `server.js` реалізує простий HTTP-сервер на Node.js, який:

- Слухає порт 3000
- Підтримує наступні маршрути:
  - `/` - головна сторінка
  - `/about` - сторінка "Про нас"
  - `/time` - виводить поточну дату і час у форматі "Поточний час: 02.10.2025, 14:30:15"
  - `/time/json` - повертає дату і час у форматі JSON
- Логує всі запити на `/time` та `/time/json` у файл `log.txt`

## Встановлення та запуск

### Вимоги

- Node.js (версія 12 або вище)

### Запуск сервера

```bash
# Перейти в папку lab4
cd lab4

# Запустити сервер
node server.js

# Або використовуючи npm скрипт
npm start
```

Після запуску сервер буде доступний за адресою: `http://localhost:3000`

## Маршрути

### GET /

Головна сторінка з навігацією до інших сторінок.

### GET /about

Сторінка "Про нас" з описом сервера.

### GET /time

Повертає HTML-сторінку з поточною датою та часом у форматі:
```
Поточний час: 02.10.2025, 14:30:15
```

### GET /time/json

Повертає JSON з поточною датою та часом:
```json
{
  "date": "02.10.2025",
  "time": "14:30:15"
}
```

**Примітка:** Кожен запит на `/time` або `/time/json` автоматично логується у файл `log.txt`.

## Файли

- `server.js` - основний файл сервера
- `log.txt` - файл з історією звернень до маршрутів `/time` та `/time/json`
- `package.json` - конфігурація npm проєкту
