# Лабораторна робота 3 - Робота з файлами в Node.js

## Мета роботи

- Ознайомитися з платформою Node.js та її можливостями
- Вивчити поняття npm (Node Package Manager) та принципи роботи з пакетами
- Навчитися створювати прості скрипти на Node.js
- Закріпити навички роботи з файловою системою (читання та запис файлів)

## Опис програми

Програма `index.js` виконує наступні дії:

1. Зчитує дані з файлу `notes.txt`
2. Вставляє на початок тексту рядок "Парний варіант — виконано студентом Скальський Володимир"
3. Зберігає результат у файл `output_notes.txt`

## Встановлення та запуск

### Вимоги

- Node.js (версія 12 або вище)
- npm (встановлюється разом з Node.js)

### Запуск програми

```bash
# Перейти в папку lab3
cd lab3

# Запустити програму
node index.js

# Або використовуючи npm скрипт
npm start
```

## Структура файлів

```
lab3/
├── index.js          # Основний скрипт програми
├── notes.txt         # Вхідний файл з даними
├── output_notes.txt  # Вихідний файл з обробленими даними
├── package.json      # Конфігурація npm проєкту
└── README.md         # Документація
```
