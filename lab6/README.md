# Лабораторна робота 6 - REST API для To-Do List

## Мета роботи

Розробити REST API для управління списком завдань (To-Do List) із використанням Express.js з підтримкою бекенду (API) та клієнтської частини (HTML + CSS + JS).

## Опис програми

REST API для управління списком завдань з наступними можливостями:

- **Структура завдання:**
  - `id` - унікальний ідентифікатор
  - `title` - назва завдання
  - `description` - опис завдання
  - `status` - статус (`active` або `done`)

- **API маршрути:**
  - `GET /api/tasks` - отримати всі завдання
  - `GET /api/tasks/:id` - отримати завдання за ID
  - `POST /api/tasks` - додати нове завдання
  - `PUT /api/tasks/:id` - змінити статус або опис
  - `DELETE /api/tasks/:id` - видалити завдання

- **Клієнтський інтерфейс:**
  - Перегляд списку завдань
  - Додавання нових завдань
  - Видалення завдань
  - Позначення завдань як виконаних
  - Фільтрація завдань (всі/активні/виконані)

- **Middleware:**
  - Логування всіх запитів до сервера у файл `requests.log`

## Встановлення та запуск

### Вимоги

- Node.js (версія 12 або вище)
- npm

### Кроки встановлення

```bash
# Перейти в папку lab6
cd lab6

# Встановити залежності
npm install

# Запустити сервер
npm start
```

Після запуску сервер буде доступний за адресою: `http://localhost:3000`

## API Документація

### GET /api/tasks

Отримати всі завдання.

**Відповідь:**
```json
[
  {
    "id": 1,
    "title": "Приклад завдання",
    "description": "Опис завдання",
    "status": "active"
  }
]
```

### GET /api/tasks/:id

Отримати завдання за ID.

**Параметри:**
- `id` - ID завдання

**Відповідь:**
```json
{
  "id": 1,
  "title": "Приклад завдання",
  "description": "Опис завдання",
  "status": "active"
}
```

### POST /api/tasks

Додати нове завдання.

**Тіло запиту:**
```json
{
  "title": "Назва завдання",
  "description": "Опис завдання"
}
```

**Відповідь:**
```json
{
  "id": 1,
  "title": "Назва завдання",
  "description": "Опис завдання",
  "status": "active"
}
```

### PUT /api/tasks/:id

Змінити статус або опис завдання.

**Параметри:**
- `id` - ID завдання

**Тіло запиту:**
```json
{
  "title": "Оновлена назва",
  "description": "Оновлений опис",
  "status": "done"
}
```

**Відповідь:**
```json
{
  "id": 1,
  "title": "Оновлена назва",
  "description": "Оновлений опис",
  "status": "done"
}
```

### DELETE /api/tasks/:id

Видалити завдання.

**Параметри:**
- `id` - ID завдання

**Відповідь:**
```json
{
  "message": "Завдання видалено",
  "task": {
    "id": 1,
    "title": "Назва завдання",
    "description": "Опис завдання",
    "status": "active"
  }
}
```
